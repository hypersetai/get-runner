# get-runner

Distribution repository for the **Hyperset Runner** binary.

Latest version: **v0.1.0**

## Install

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/hypersetai/get-runner/main/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/hypersetai/get-runner/main/install.ps1 | iex
```
